    Section 9.3 Chi-square Association Test

    In this section we will discuss the chi-square test for association between two categorical variables and the chi-square test for homogeneity of proportions. The chi-square test for association is used to determine whether there is a significant relationship between two categorical variables in a population. The chi-square test for homogeneity of proportions is used to compare the distribution of a categorical variable across different populations or groups. Both tests involve comparing observed counts to expected counts under the assumption of independence or homogeneity, and calculating a chi-square statistic to assess the significance of any observed differences.
    🔗

    Objectives

    At the end of this section you will be able to:
    🔗
    1. Test for an association between two categorical variables using the chi square test.
      🔗
      🔗
    2. Test for the homogeneity of multiple proportions using the chi square test.
      🔗
      🔗
    🔗

    Subsection 9.3.1 Understanding the Basics

    A chi-square test for two categorical variables is used to determine whether there is an association between the variables in a population. This test is applied to data arranged in a two-way table, where the counts show how individuals fall into combinations of categories. The test compares the observed counts in the table to the counts we would expect if the two variables were independent. If the observed counts differ enough from the expected counts, we have evidence of an association between the two categorical variables.
    🔗
    The null hypothesis will be a statement that the variables are independent of each other or that there is no association between them. The alternative hypothesis will be a statement that the variables are dependent or there is an association between them.
    🔗
    Since we are working with a two-way table, we need to find the expected count for each cell. It is found by taking the total for the row multiplied by the total for the column and then divided by the grand total which is the total sample size.
    🔗
    The expected count for a cell is
    \begin{equation*} E=\dfrac{(\text{row total})(\text{column total})}{\text{table total}} \end{equation*}
    🔗
    The test statistic is the difference between the observed and expected value for each cell, squared, and divided by the expected value. Then we find the sum of the values. The formula is the same as the chi-square goodness-of-fit test. But we much be careful to pair up each cell. The formula for the test statistic is
    \begin{equation*} \chi^2=\sum \frac{(O-E)^2}{E} \end{equation*}
    🔗
    When the test statistic is small, that indicates there is little change in the observed and expected values thus there is no association and the variables are independent of each other. If the change is larger then there is an association and the variables are dependent on one or more of the categories.
    🔗

    Definition 9.3.1.

    A chi-square test for association is used to determine whether two categorical variables are associated in a population. The data are arranged in a two-way table, and the observed counts in each cell are compared to the expected counts that would occur if the variables were independent. The expected count for a cell is \(E=\dfrac{(\text{row total})(\text{column total})}{\text{table total}}\text{,}\) and the test statistic is \(\chi^2=\sum \frac{(O-E)^2}{E}\text{,}\) where \(O\) is the observed count and \(E\) is the expected count. The degrees of freedom for the test are \(df=(r-1)(c-1)\text{,}\) where \(r\) is the number of rows and \(c\) is the number of columns in the table.
    🔗
    🔗
    🔗

    Subsection 9.3.2 Example for the chi-square independence test

    For this course, the chi-square test for independence of two categorical variables is equivalent to the chi-square test for an association between two categorical variables.
    🔗
    A biologist wants to determine whether plant growth is independent of fertilizer type. A sample of \(90\) plants is randomly assigned to one of three fertilizer types, and after a fixed growing period each plant is classified as having low, medium, or high growth.
    🔗
    At the \(0.05\) significance level, test whether plant growth category is independent of fertilizer type.
    🔗

    Example 9.3.3. Plant Growth.

    At the \(0.05\) significance level, test whether plant growth category is independent of fertilizer type. Include the hypotheses, expected values table, chi-square test and p-value, and the conclusion in context.
    🔗
    Solution 1. Hypotheses
    This is a chi-square test of independence between two categorical variables.
    🔗
    The hypotheses are:
    🔗
    \(H_0:\) plant growth category and fertilizer type are independent,
    🔗
    \(H_a:\) plant growth category and fertilizer type are dependent.
    🔗
    🔗
    Solution 2. Table of Values
    Since we are assuming the variables are independent of each other, we would expect the counts to be the same for each category of plant growth and for each type of fertilizer.
    🔗
    First, we find the row totals for how many plants for each fertilizer, column totals for growth, and the grand total for the number of plants in the study.
    🔗
    Under the assumption of independence, each expected count for each cell in the table is
    \begin{equation*} E=\dfrac{(\text{row total})(\text{column total})}{\text{grand total}}\text{.} \end{equation*}
    🔗
    For Fertilize A with Low growth, the expected value is \(\frac {30 \times 30}{90}=10\)
    🔗
    For Fertilize A with Medium growth, the expected value is \(\frac {30 \times 33}{90}=11\)
    🔗
    For Fertilize A with High growth, the expected value is \(\frac {30 \times 27}{90}=9\)
    🔗
    For Fertilize B with Low growth, the expected value is \(\frac {30 \times 30}{90}=10\)
    🔗
    For Fertilize B with Medium growth, the expected value is \(\frac {30 \times 33}{90}=11\)
    🔗
    For Fertilize B with High growth, the expected value is \(\frac {30 \times 27}{90}=9\)
    🔗
    For Fertilize C with Low growth, the expected value is \(\frac {30 \times 30}{90}=10\)
    🔗
    For Fertilize C with Medium growth, the expected value is \(\frac {30 \times 33}{90}=11\)
    🔗
    For Fertilize C with High growth, the expected value is \(\frac {30 \times 27}{90}=9\)
    🔗
    Now we can organize the values into the Expect Counts Table.
    🔗
    🔗
    Solution 3. Chi-square Test
    Now we compute the test statistic pairing each cell:
    \begin{equation*} \chi^2=\sum \frac{(O-E)^2}{E}\text{.} \end{equation*}
    🔗
    The contributions for fertilizer A are: \(\frac{(14-10)^2}{10}=1.60,\quad \frac{(10-11)^2}{11}\approx 0.091,\quad \frac{(6-9)^2}{9}=1.00\)
    🔗
    The contributions for fertilizer B are: \(\frac{(9-10)^2}{10}=0.10,\quad \frac{(12-11)^2}{11}\approx 0.091,\quad \frac{(9-9)^2}{9}=0.00\)
    🔗
    The contributions for fertilizer C are: \(\frac{(7-10)^2}{10}=0.90,\quad \frac{(11-11)^2}{11}=0.00,\quad \frac{(12-9)^2}{9}=1.00\)
    🔗
    So, \(\chi^2 \approx 1.60+0.091+1.00+0.10+0.091+0.00+0.90+0.00+1.00=4.782 \text{.}\)
    🔗
    The degrees of freedom is found be take one from the number of rows, and one from the number of columns, then we multiply the two values. There are 3 rows for fertilizer types and 3 columns for growth categories. So the degrees of freedom is:
    \begin{equation*} df=(r-1)(c-1)=(3-1)(3-1)=(2)(2)=4\text{.} \end{equation*}
    🔗
    Using a chi-square distribution with \(4\) degrees of freedom for a right tailed test, the \(p\)-value is \(p=0.310\text{.}\)
    🔗
    🔗
    Solution 4. Conclusion
    Since \(p=0.31 \gt 0.05\text{,}\) we fail to reject \(H_0\text{.}\)
    🔗
    There is not sufficient evidence at the \(0.05\) significance level to conclude that plant growth category and fertilizer type are dependent.
    🔗
    🔗
    🔗
    When there is a small chi-square test value, there is little difference between the observed and expected values thus we do not reject the null hypothesis and assume they are independent or that there is no association. When the chi-square value is large, there is a difference between the observed and expected values thus we reject the null hypothesis and support the alternative hypotheses that the variables are dependent or associated with each other.
    🔗

    🔗
    Press [2nd] [\(x^{-1}\)] for the Matrix, use the arrow to the right twice for the EDIT menu, we are going to select matrix 1: A and press [ENTER].
    🔗
    We need to identify the size of the matrix, input 3 rows (across/horizontal) x 3 columns (down/vertical) and then input the observed data by hitting [ENTER] after each entry. The data is entered across each row.
    🔗

    🔗

    🔗
    Identify where the observed matrix is [A] and the expected matrix is [B] which the calculator will calculate for you. Scroll to Calculate and press [ENTER] to perform the test.
    🔗
    The statistical tests value is \(\chi^2 = 4.782\) and a p-value of \(p=0.310\) with 4 degrees of freedom.
    🔗

    Olympic Medals
    🔗
    A sports analyst wants to know whether the type of medal won at the 2024 Paris Olympics was independent of country for five countries with the most total medals. The top five countries by total medals were the United States, China, Great Britain, France, and Australia.
    🔗
    At the \(0.05\) significance level, test whether medal type (gold, silver, or bronze) is independent of country.
    🔗

    Checkpoint 9.3.13. Olympic Medals.

    At the \(0.05\) significance level, test whether medal type (gold, silver, or bronze) is independent of country. Include the hypotheses, expected values table, chi-square test and p-value, and the conclusion in context.
    🔗
    Solution 1. Hypotheses
    This is a chi-square test of independence. Although the sample is not random, there are 5 in each category. We could have selected 5 random countries, but not all countries had at least five medals.
    🔗
    The hypotheses are:
    🔗
    \(H_0:\) Medal type and country are independent.
    🔗
    \(H_a:\) Medal type and country are dependent.
    🔗
    🔗
    Solution 2. Expected Values
    Compute the expected counts for each using the totals:
    \begin{equation*} E=\dfrac{(\text{row total})(\text{column total})}{\text{grand total}}. \end{equation*}
    🔗
    The row totals are \(126\text{,}\) \(91\text{,}\) \(65\text{,}\) \(64\text{,}\) and \(53\text{.}\) The column totals are \(128\) gold, \(138\) silver, and \(133\) bronze, with grand total \(399\text{.}\)
    🔗
    🔗
    Solution 3. Statistical Test
    Next compute the test statistic using each cell:
    \begin{equation*} \chi^2=\sum \dfrac{(O-E)^2}{E}\text{.} \end{equation*}
    🔗
    Using the observed and expected counts, the chi-square test statistic is \(\chi^2 \approx 12.65\text{.}\)
    🔗
    The degrees of freedom using the 5 country rows and 3 medal columns as
    \begin{equation*} df=(r-1)(c-1)=(5-1)(3-1)=8\text{.} \end{equation*}
    🔗
    For a chi-square distribution with \(8\) degrees of freedom, the \(p\)-value is approximately \(0.125\text{.}\)
    🔗
    🔗

    🔗
    The degrees of freedom are \((r-1)(c-1)=(3-1)(5-1)=8.\)
    🔗
    Using a chi-square distribution with \(8\) degrees of freedom, the \(p\)-value is \(p=0.000014\text{.}\)
    🔗
    🔗
    Solution 4. Conclusion
    Since \(p \lt 0.05\text{,}\) we reject \(H_0\text{.}\)
    🔗
    There is sufficient evidence at the 0.05 significance level to conclude that most-used social media platform is associated with age group.
    🔗
    In context, the pattern of platform use differs across the three age groups. Younger adults are more likely to report Instagram and TikTok as their most-used platforms, while older adults are more likely to report Facebook and YouTube.
